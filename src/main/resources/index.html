Say hello to spray-geoip!

The GeoIp lookup rest service.

Usage

Get the lookup data in json format: 

$ curl https://damp-earth-2108.herokuapp.com/geoip/51.1.43.15
{"latitude":51.5,"timezone":"Europe/London","countryName":"United Kingdom","longitude":-0.1300048828125,"countryCode":"GB"}
